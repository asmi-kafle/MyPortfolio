Operating Systems

In this course we studied the general architecture and the most critical components of operating systems. The objective of this course is to make you familiar with the general architecture and the most critical components of operating systems. In this course, we implemented aspects of oeprating systems, such as memory management, file systems, threading, scheduling and resource mamagement in general. We also learnt fundamental approaches to virtualization and we learnt to support distributed systems. We uses case studies to learn more about realization approaches. And ended the course by implementing a simple operating system that can be booted on a bare PC.

Computer Design and Architecture

In this course, we learnt about and used register transfer languages and simulation tools to describe and simulate computer operation. We also learnt about central processing unit organization, microprogramming, input/output and memory system architectures.

We used c++ programming language for the projects.

Computer Networks and System
